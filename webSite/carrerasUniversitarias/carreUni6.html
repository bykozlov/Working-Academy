Ingeniero de Sitemas Urgente/Cercado de Lima

Corporación Arzobispo Loayza SAC

Lima

Importante empresa del sector educativo, se encuentra en búsqueda de un Ingeniero(a) de Sistemas

REQUISITOS:
  • Bachiller o Titulado de Ingeniería de Sistemas, Ingeniería Informática, Ciencias de la Computación o afines.
  • Experiencia mínima de un año en el puesto o posiciones afines.
  • Disponibilidad de inicio inmediato y presencia en el Cercado de Lima.

    • CONOCIMIENTOS:
  • Dominio de HTML, CSS y Javascript para desarrollo de landings, webs.
  • Backend con PHP (Intermedio) / Codeigniter u otros frameworks (Deseable).
  • Experiencia en base de datos SQL Server (Intermedio)/ MySQL (Deseable, pero no obligatorio).
  • Conocimiento de sistemas de control de versiones (Deseable, pero no obligatorio).
  • Conocimientos básicos de metodologías de desarrollo.

    • COMPETENCIAS:
  • Trabajar en equipo, aprendizaje constante, compromiso, responsable, proactivo.

    • FUNCIONES:
  • Evaluar la situación actual de la estructura de la base de los datos existente en la Institución y realizar Copia de Seguridad (Backup) de dicha información.
  • Mantener el servicio de comunicación y trasmisión de los datos (RED LAN) en buen estado.
  • Ingresar a los archivos de cada uno de los administrativos, para verificar si no se ha cometido errores en el empleo de los respectivos sistemas

    • Beneficios:
  • Lo que por ley corresponda
  • Pagos mensuales
